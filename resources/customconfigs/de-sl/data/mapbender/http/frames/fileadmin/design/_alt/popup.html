WMS Zusammenfassung The spatial data hosted by this service is dual licensed as GNU FDL and PGL. All data can be used, copied, modified and shared free of any license fees. Keep in mind that this data is NOT UNlicensed! Refer to http://www.gnu.org/ for the complete license text. The data contained in this project covers the topography, transport infrastructure, settlements, places and postal codes of Germany. The data is fairly old and needs to be updated. You are welcome to join. Do not use this data for navigation purposes or any application where potentially lives might depend on. This service is brought to you by CCGIS (http://www.ccgis.de). This service runs a UMN MapServer (http://www.umn-mapserver.de) a PostgreSQL database (http://www.postgresql.org) which is spatially enabled with PostGIS (http://postgis.refractions.ca). You can edit the data using Mapbender (http://www.mapbender.org) or any other OGC WFS-T digitizing client.
