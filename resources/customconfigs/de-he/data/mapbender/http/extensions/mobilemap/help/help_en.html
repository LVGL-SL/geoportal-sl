Layer control:
You can change the baselayers by taping on the radio buttons or names. You can only enable one baselayer.
The other layers (overlays) can be checked or unchecked with the corresponding checkboxes.
If you want to query an object of a layer, activate the "info-option" and make a single tap on the object.
Gestures for touchdevices:
Pinch-Zoom: allows you to zoom in or out by moving two fingers farther apart or closer together while touching the display. (Note: Android 2.x does not support this gesture!)
Double Tap - Zoom: using a double tab on the map you can zoom in to the desired point.
Pan: "hold" on the map after a simple tap and simultaneously moving the map you can move (pan) freely
Single Tap: with a single tap you can query the active layer (see layer control)
